About Me

Hey there, my name is Emmanuel. I'm an aspiring backend web developer!

I had my first exposure to the "behind-the-scenes" of the internet, sometime in 2020, when I had the need to create a website for an eBook I had written at the time.

I started out with WordPress, but it was a bit overwhelming for me. So I tried out the Shopify platform.

Notwithstanding, cost considerations drove me back to WordPress. Eventually, I managed to set up a descent WordPress website for my eBook.

Even so, I wasn't completely satisfied with the plugin ecosystem of WordPress. I wanted more design freedom and flexibility.

My goal is to become a highly sought-after backend engineer.
